# CBETA Reader - 全藏研讀空間與未來發展藍圖 (Future Architecture Roadmap)

## 📌 願景與定位
將 CBETA Reader 從「輕量個人隨身閱讀器」逐步升級為「支援完整 CBETA 大藏經（10,000+ 部經典、數 GB 級資料）的專業佛學研讀空間」，提供兼具輕盈體驗與深遠研讀能力的數位閱讀環境。

---

## 🛠️ 四大架構設計維度

### 1. 儲存與架構策略：離線與串流雙軌機制 (Hybrid Storage)
- **全藏線上串流 (Cloud Streaming Mode)**：隨讀隨載（On-Demand），點擊經典時極速取得該卷資料並自動快取，無須預先占用硬碟。
- **按部類離線下載包 (Volume & Canon Bundles)**：提供「按部類一鍵下載」（如《阿含部》、《般若部》、《印順導師著述》），並採用高壓縮比格式（MessagePack / LZ-String）。
- **智慧 LRU 清理 (Smart Storage Eviction)**：設定容量上限，自動清理長時間未閱讀的舊經文快取，**永久保留讀者的所有劃線、筆記、書籤與設定**。
- **高效能瀏覽器資料庫 (OPFS / SQLite-wasm)**：提供 GB 級資料的毫秒級跳轉與高速檢索。

### 2. UI/UX 空間規劃：「藏書閣」與「大圖書館」雙視角
- **個人藏書閣 (My Shelf)**：放置讀者研讀、劃線、精選的經文（10~100 部），保持極簡、無溫度的無壓力氛圍。
- **大大圖書館 (Grand Library)**：完整收錄 CBETA 浩瀚目錄。
  - 四層級導航：`藏` $\rightarrow$ `部` $\rightarrow$ `經` $\rightarrow$ `卷`。
  - 支援 CBETA 標準經典編號（如 `T0262`、`T1545`）瞬間跳轉。
  - 全面導入**虛擬滾動列表 (Virtual Scrolling)** 確保 60 FPS 順暢體驗。

### 3. 全藏級別全文檢索 (Full-Text Search Engine)
- **分層檢索**：本地藏書採 Client-side 檢索；全藏採 Serverless API / Edge Worker 0.1 秒極速傳回命中結果。
- **進階過濾**：限定部類檢索、同譯本與同本異譯檢索。

### 4. 專業研讀與學術工具箱 (Professional Study Tools)
- **雙欄對照模式 (Parallel Reading)**：支援左右雙欄同本異譯比對（如羅什譯 vs 玄奘譯）、經文與導師講記/古德論疏連動對照與同步滾動。
- **佛學辭典與懸停校勘**：整合《丁福保佛學大辭典》、《三藏法數》，專有名詞選取即時查閱，校勘符號懸停浮動顯示。
- **學術引用格式產生器 (Academic Citation)**：劃線複製自動附帶標準 CBETA 引用格式（冊號、經號、卷次、頁碼欄位）。
- **研讀地圖與進度統計**：視覺化展示大藏經閱讀涵蓋率與個人筆記分佈。

---

## 📅 漸進式發展階段 (Progressive Enhancement)

1. **Phase 1（當前階段）**：專注於輕量隨身閱讀、極致字體與單書劃線重點備份還原。
2. **Phase 2**：推出「大圖書館」四級選單與「部類按需串流/離線包」。
3. **Phase 3**：引進雙欄經論對照、佛學辭典、全藏檢索與學術引用工具。
